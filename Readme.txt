Instructions:

1. Run the file "xampp-win32-1.7.4-VC6-installer.exe" file and install xampp(if not installed already).
2. Copy the "highway_runners" folder in the following directory: "C:/xampp/htdocs/"
3. Run "localhost/phpmyadmin" in your browser and import the database from the "localhost.sql" file.
4. To run the website, enter the URL as follows: "localhost/highway_runners/"
